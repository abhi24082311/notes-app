A simple Notes application built using Spring Boot and Spring Data JPA.
It provides REST APIs to create, read, update, and delete notes.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 / PostgreSQL
- Maven

## API Endpoints
- GET /notes
- POST /notes
- PUT /notes/{id}
- DELETE /notes/{id}

## How to Run
- mvn clean package
- java -jar target/*.jar
